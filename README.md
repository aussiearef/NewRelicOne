# New Relic: From Beginner to Advanced

Welcome to the repository for the online course "New Relic: From Beginner to Advanced." This course is designed to introduce you to the fundamentals of New Relic One and gradually advance your skills to an advanced level.

[![New Relic One Course](https://img-c.udemycdn.com/course/750x422/1733814_8701_4.jpg)](https://www.udemy.com/course/new-relic-apm-application-performance-management-for-devops/?referralCode=E3AB7C6B7055A4C4656C)

## Course Overview

In this course, you will learn:

- Instrumenting applications, databases, cloud environments and infrastrucutre with New Relic Agent.
- Collecting telemetry signals with OpenTelemetry.
- Creating alerts and notifications to Teams, Slack, etc.
- New Relic's Query Language (NRQL) for querying your data.
- Monitoring AI (Large Language Models) e.g., Token usage and Performance.
- Using New Relic's AI capabilities.
- Administration tasks, cost management and so much more.
